## Installation

1. Install theme from the [Marketplace]()
2. Go to `File > Preferences > Color Theme`
3. Select `Zen Dark`. 

Alternatively install via CLI:
```
code --install-extension miguelsolorio.min-theme
```

**Enjoy!**
